# Скрипт для быстрого запуска настроенного tmux

## Установка

1. Скачать скрипт и файл `tms.config`
2. Добавить их папку своих глобально исполняемых скриптов
3. Сделать файл исполняемым: `sudo chmod +x tms`
4. Добавить в `.bashrc` или `.zshrc`, если Oh-my-ZSH: `export PATH="$HOME/my_scripts:$PATH"`
5. Добавить в файл `tms.config` свои значения для переменных, которые будут использоваться при запуске

## Использование

1. `tms` - запускает настроенный tmux
2. `tms -k` или `tms --kill` - убивает сессию tmux
